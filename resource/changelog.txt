Freelance Office is a web based client portal application that you can use to manage your clients in one click.<br>

### What is needed to Install? ### <br>
<ul>
<li> * Apache Server </li>
<li> * MySQL 5 and above</li>
<li> * Mod_Rewrite enabled</li>
<li> * Cron Jobs</li>
<li> * CURL PHP Library</li>
<li> * PostMark Account (Optional)</li> 
</ul>
<br>
